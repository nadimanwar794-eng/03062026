# IIC Study App (NSTA)

A full-featured student learning platform — Lucent book page viewer, MCQ practice, TTS reading, homework, revision hub, and admin dashboard — ported from Vercel to Replit.

## Run & Operate

- `artifacts/iic-study-app: web` workflow — main web app (port 22698, previewPath `/`)
- `pnpm run typecheck` — full typecheck across all packages
- Firebase backend: uses the project configured in `artifacts/iic-study-app/src/firebase.ts`

## Stack

- pnpm workspaces, Node.js 20, TypeScript 5.x
- Frontend: React 18 + Vite, Tailwind CSS, Framer Motion, wouter
- Backend: Firebase (Firestore, Realtime Database, Auth, Storage)
- PWA: vite-plugin-pwa (service worker, offline support)
- AI: Groq/Gemini stubs (disabled; stubs in `src/services/groq.ts`)

## Where things live

- `artifacts/iic-study-app/src/App.tsx` — root component; state machine driving all views (BOARDS → SUBJECTS → CHAPTERS → LESSON)
- `artifacts/iic-study-app/src/firebase.ts` — Firebase config + all Firestore/RTDB helpers
- `artifacts/iic-study-app/src/components/StudentDashboard.tsx` — main student experience (Lucent book, homework, revision, MCQ)
- `artifacts/iic-study-app/src/components/AdminDashboard.tsx` — content management for admins
- `artifacts/iic-study-app/src/services/groq.ts` — AI stubs (Groq/Gemini removed; safe no-ops)
- `artifacts/iic-study-app/src/utils/` — credit system, level system, subscription, storage helpers

## Architecture decisions

- All routing is state-based (not URL-based); `view` field in `AppState` drives what's rendered.
- Firebase is the single source of truth for content, users, and settings — no separate API server needed.
- AI generation (Groq/Gemini) is stubbed out; admin-uploaded content in Firebase is served instead.
- PWA service worker is registered for offline reading; `dev-dist/sw.js` is auto-generated by vite-plugin-pwa.
- `tempSelectedChapter` + `handleContentGeneration` follow a set-then-call pattern; the modal open path (`setShowPremiumModal`) ensures state is settled before content generation runs.

## Product

Students log in (email/mobile/anonymous via Firebase Auth), browse boards → subjects → chapters, and access reading notes (chunked TTS), write-mode HTML notes, MCQ practice, PDF, video, and audio content per chapter. Admins upload and manage all content via an in-app dashboard. Subscription tiers (Free/Basic/Ultra) gate daily limits and premium content.

## Gotchas

- Firebase Firestore "permission-denied" errors on fresh load are expected in anonymous mode; the app falls back to localStorage cache and continues to function.
- Do not run `pnpm dev` at the workspace root — it has no dev script. Use the managed workflow `artifacts/iic-study-app: web`.
- `dev-dist/sw.js` revision hash changes on every cold start; exclude from meaningful diffs.
- `.migration-backup/` contains the original imported zip and extracted files — do not delete.
